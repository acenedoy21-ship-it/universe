# GitUniverse

Free browser-based space simulator. Explore a galaxy map, simulate gravity and fluid physics, view realistic skies with atmospheric scattering, and learn with guided lessons.

## Features

- N-body gravity simulation with real physics
- Procedural planet generation
- Galaxy map with real star systems and deep-sky objects
- Atmospheric scattering sky view
- Guided physics lessons
- Fluid/SPH simulation
- Habitable zone visualization
- Historical date navigation

## Running Locally

```bash
# Start the local server (required for SharedArrayBuffer support)
python server.py 8000

# Open in browser
# http://localhost:8000
```

Or double-click `start.bat` on Windows.
